# _Word Blast_

#### By _**Mike Donovan, Chris Johnedis, Viradeth Xayananh, Charles Cookman**_

#### _A timed typing game with a different style for each level_

## Technologies Used

* _VS Code_
* _Git and GitHub_
* _Gimp_
* _languages HTML, CSS, Javascript, Markdown, Mermaid_

## Description

_This game uses words that have to be typed set up to a timer where if they are not completed in time damage your health bar, as you complete words you collect points to your score._

## Setup/Installation Requirements

* _Visit our repository here: https://github.com/eggborne/team-project and clone it._
* _After Cloning to your own device in the terminal of your directory run "$ npm install" in the terminal to set up the Node Packet Manager_
* _in the Game.js file set the this.level to the level you wish to play, levels are listed in the this.levels array and are numbered respectively. You may start on any level as they transfer after completion_
* _in the terminal run "$ npm run start" to start a local server if it doesn't load a new tab or web broser page go to your web browser and in the URL put 'localhost:8080'_
* _It should start and begin the game by pressing space and type the words as you see them_

_If you are not familiar with npm please review basics if you have any trouble_
